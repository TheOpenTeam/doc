## 存储类型
| Java 类型 | Rust 类型 | Minecraft 存储方式 |
|-----------|-----------|-------------------|
| `boolean` | `bool` | 1 字节，`0x01` 为 true，`0x00` 为 false |
| `byte` | `i8` | 1 字节，有符号 |
| `short` | `i16` | 2 字节，大端序 |
| `int` | `i32` | 4 字节，大端序 |
| `long` | `i64` | 8 字节，大端序 |
| `float` | `f32` | 4 字节，大端序 IEEE 754 |
| `double` | `f64` | 8 字节，大端序 IEEE 754 |
| `String` | `String` | VarInt 长度前缀 + UTF-8 字节 |
| `VarInt` | `i32` | 1-5 字节，变长编码 |
| `VarLong` | `i64` | 1-10 字节，变长编码 |
| `UUID` | `[u8; 16]` 或 `uuid::Uuid` | 16 字节，大端序 |
| `Position` | 自定义结构体 | 64 位打包：x(26位) + z(26位) + y(12位) |
| `Angle` | `u8` | 1 字节，旋转角 (360/256 度) |
| `Byte Array` | `Vec<u8>` | VarInt 长度前缀 + 字节数据 |
| `NBT Tag` | `nbt::Value` | 二进制 NBT 格式 |
| `Optional<T>` | `Option<T>` | 布尔标志 + 存在时的 T |
| `Slot` | 自定义结构体 | 物品数据，格式较复杂 |
| `Entity Metadata` | 自定义结构体 | 特定格式的元数据列表 |
