# 蚯蚓语教程

重制版 1.02

## 简要

在本书的蚯蚓语介绍中，对于例子，我们以“e.g.”开头，其内部的**核心词汇加粗**处理。

蚯蚓语是极具特色的语言。

## 一、核心词汇

蚯蚓语的核心词汇主要有以下汉字：

- 赖
- 毁
- 错
- 完
- *慌*

## 二、核心语法

### 陈述句

蚯蚓语的陈述句由以下成分组成：

1. 主语
2. 状语(可省略)
3. **核心词汇**
4. 陈述语气词（通常为“**了**”）

***e.g.*** 我**错**了 你**完**了 我**毁**了 你**毁**了 你全**毁**了

### 祈使句

蚯蚓语的祈使句由以下成分组成：

1. 状语
2. **核心词汇 / 动词**
3. 宾语
4. **任意类型的**语气词

其中，状语、宾语和语气词都可以省略。

***e.g.*** **赖**你 **赖**了 **毁**了 **慌**什么 全**赖**你 都**完**了

也有单字的祈使句：

***e.g.*** **赖**

### 疑问句

蚯蚓语的疑问句分为两种：

- 普通疑问
- 祈使疑问（结合两种句型的特点）
- “没”疑问

#### 普通疑问

普通疑问由以下成分组成：

1. 陈述句
2. (了)吗

***e.g.*** 你毁了吗 你错了吗

#### 祈使疑问

祈使疑问由以下成分组成：

1. **祈使句**
2. (了)吗

***e.g.*** **错了**吗 **赖**吗 **完了**吗 **毁了**吗

#### “没”疑问

“没”疑问由以下成分组成：

1. 主语(可以省略)
2. **核心词汇**
3. 没
4. 核心词汇

***e.g.*** **错**没**错** **毁**没**毁** **完**没**完**

#### 特殊疑问句

特殊疑问句由以下成分组成：

1. 核心词汇
2. 特殊疑问词（如谁）

***e.g.*** **赖**谁

### 待定成分句

当一个人发出核心词汇，但并不构成完整句子（比如单字祈使）时，与下一个人的回答构成待定成分句。

***e.g.***

A：赖

此时B的回答既可以是复述核心词汇的完整句，

B：**赖**瑞琪

也可以回答包括其他核心词汇的句子，只要句子里指定了对象，

B：瑞琪**毁**了

## 三、征求意见

征求意见的词汇有以下：

- 交易
- 成交
- 不成交

### 交易句

交易用于请求对方做一个动作。

完整交易句的结构：

1. 交易
2. 任意动宾结构
3. 语气词“吗”（可省略）

***e.g.*** **交易**造刷铁机

而在日常交流中，使用简单交易句的情景也很多。

简单交易句的结构：

1. 交易
2. 名词
3. 语气词“吗”（可省略）

***e.g.*** **交易**mc吗 **交易**刷铁机

为了句意连贯，在使用交易句后通常可以再使用“成交吗”。

### 回答

交易句的回答通常具有肯定、否定两面。

肯定回答，直接回答成交。否定回答，直接回答不成交。

若在刚刚已经回答过交易句，并且以“成交”作为答案，若这次回答仍然为“成交”，为了句意连贯，通常使用“也成交”。

## 四、其他词汇

随着时间，许多蚯蚓语者的常用词汇演变为蚯蚓语的一部分。

### 绷

“绷”通常用于一个人的言论不合常理时，表示委婉的无语、尴尬之心。通常单用。

### 状态询问

当他人称呼一个人时，询问事宜可以采取两种方法：

- 怎么了
- 说人话

后者的语气较为强烈，通常表达厌烦、不满心，而前者通用。

### 人格表述

可以用以下词汇形容一个人：

- 人机
- 傻逼
- 唐（糖）
- 耳鼻

注意的是，这些词语都是含有极强的贬义色彩，不得随意使用。

## 五、缩写

为了方便交流，蚯蚓语可以通过拼音缩写的方式来表达，但以下形式不行：

1. 单字祈使句或者只有一个核心词汇
2. 该句不是蚯蚓语

***e.g.*** 你完了: nwl 我错了: wcl 赖你: ln

特别地，“也成交”的缩写形式是 yecj / ycj
